# Project Structure

Key directories and files:

- `src/`: application source.
- `src/routes/`: file-based routes (TanStack Router).
- `src/components/`: reusable UI building blocks.
- `src/content/` and `src/data/`: site content and data helpers.
- `src/db/`: Drizzle schema and database helpers.
- `src/integrations/`: external services (WorkOS, etc.).
- `public/`: static assets.
- `manual/`: this mdBook documentation.
- `.gemini/`: Gemini extension configuration.

Generated files:

- `src/routeTree.gen.ts` is generated by TanStack Router.
